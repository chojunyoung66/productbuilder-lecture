# 오늘 뭐 먹지? - 저녁 메뉴 선택기

## 개요

"오늘 뭐 먹지?"는 사용자가 버튼을 클릭하면 저녁 메뉴를 랜덤으로 추천해주는 간단하고 재미있는 웹 애플리케이션입니다. 현대적인 웹 기술을 사용하여 별도의 프레임워크 없이 제작되며, 깔끔하고 직관적인 사용자 경험을 제공하는 데 중점을 둡니다.

## 디자인 및 기능 명세

### 1. 스타일 및 디자인
- **색상 팔레트:** 편안하고 현대적인 색상 조합을 사용합니다.
- **폰트:** 가독성이 좋은 웹 폰트를 적용합니다.
- **레이아웃:** 모바일과 데스크탑을 모두 지원하는 반응형 중앙 정렬 레이아웃을 사용합니다.
- **애니메이션:** 버튼 클릭 시 메뉴가 추천되는 과정에 간단한 애니메이션 효과를 추가하여 시각적 즐거움을 더합니다.

### 2. UI 컴포넌트
- **메인 컨테이너:** 앱의 모든 요소를 감싸는 컨테이너입니다.
- **헤더:** "오늘 뭐 먹지?"라는 앱 제목을 표시합니다.
- **메뉴 추천 버튼:** 사용자가 클릭하여 메뉴 추천을 시작하는 기본 상호작용 요소입니다.
- **결과 표시 카드:** 추천된 메뉴 이름이 표시되는 영역입니다. 처음에는 보이지 않다가 메뉴가 선택되면 나타납니다.

### 3. 핵심 기능
- **메뉴 목록:** JavaScript 파일 내에 다양한 저녁 메뉴 목록을 배열로 저장합니다.
- **랜덤 메뉴 선택:** 사용자가 버튼을 클릭하면 저장된 메뉴 목록에서 무작위로 하나의 메뉴를 선택합니다.
- **결과 업데이트:** 선택된 메뉴 이름을 결과 표시 카드에 동적으로 업데이트하고 사용자에게 보여줍니다.
